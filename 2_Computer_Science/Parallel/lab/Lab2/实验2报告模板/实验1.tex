\section{实验目的}
\begin{itemize}
   \item 掌握 MPI 程序的编译和运行方法。
    \item 理解 MPI 点对点通信的基本原理。
    \item 了解 MPI 程序的 GDB 调试流程。
\end{itemize}

\section{实验内容}
\begin{itemize}
   \item 使用 MPI 点对点通信实现并行矩阵乘法。
    \item 设置进程数量（1$\sim$16）及矩阵规模（128$\sim$2048）。
    \item 根据运行时间，分析程序的并行性能。
\end{itemize}

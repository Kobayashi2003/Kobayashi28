VNDB API v3 Savedata Routes

List Savedata

GET /api/v3/savedata

Description: Get a list of savedata

Query Parameters:

  • vn_id: VN ID to filter savedata
  • page: Page number (default: 1)
  • limit: Number of items per page (default: 20, max: 100)

Get Savedata

GET /api/v3/savedata/:id

Description: Get a specific savedata

Upload Savedata

POST /api/v3/savedata

Description: Upload savedata for a VN

Body: Form-data with 'files[]' containing savedata files

Query Parameters:

  • vn_id: VN ID for the savedata

Update Savedata

PUT /api/v3/savedata/:id

Description: Update a specific savedata

Body: Form-data with updated savedata file

Delete Savedata

DELETE /api/v3/savedata/:id

Description: Delete a specific savedata

Delete All Savedata

DELETE /api/v3/savedata

Description: Delete all savedata for a VN

Query Parameters:

  • vn_id: VN ID to delete all savedata for

Download Savedata

GET /api/v3/savedata/download/:id

Description: Download a specific savedata or all savedata for a VN

Note: If :id is a savedata ID, it will download a single file. If :id is a VN ID, it will download a zip of all savedata for that VN.

VNDB API v3 Overview

Base URL

All API requests should be made to: /api/v3

Resources

  • Database: /api/v3/database
  • Images: /api/v3/images
  • Savedata: /api/v3/savedata
  • Search: /api/v3/search

Authentication

Authentication details to be determined. All endpoints currently do not require authentication.

Response Format

All responses are returned in JSON format. Successful responses will include a data field containing the requested information. Error responses will include an error field with details about the error.

HTTP Status Codes

  • 200 OK: The request was successful
  • 201 Created: The request was successful and a resource was created
  • 204 No Content: The request was successful but there is no representation to return (i.e. the response is empty)
  • 400 Bad Request: The request could not be understood or was missing required parameters
  • 404 Not Found: Resource was not found
  • 500 Internal Server Error: An unexpected error occurred on the server

Rate Limiting

Rate limiting details to be determined. Please use the API responsibly.

VNDB API v3 Image Routes

List Images

GET /api/v3/images

Description: Get a list of images

Query Parameters:

  • type: Image type (vn or character)
  • id: VN or character ID
  • page: Page number (default: 1)
  • limit: Number of items per page (default: 20, max: 100)

Get Image

GET /api/v3/images/:id

Description: Get a specific image

Upload Images

POST /api/v3/images

Description: Upload images for a VN or character

Body: Form-data with 'files[]' containing image files

Query Parameters:

  • type: Image type (vn or character)
  • id: VN or character ID

Update Image

PUT /api/v3/images/:id

Description: Update a specific image

Body: Form-data with updated image file

Delete Image

DELETE /api/v3/images/:id

Description: Delete a specific image

Delete All Images

DELETE /api/v3/images

Description: Delete all images for a VN or character

Query Parameters:

  • type: Image type (vn or character)
  • id: VN or character ID

Download Images

GET /api/v3/images/download

Description: Download images for a VN or character

Query Parameters:

  • type: Image type (vn or character)
  • id: VN or character ID

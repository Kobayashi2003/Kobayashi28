VNDB API v3 Database Routes

List Resources

GET /api/v3/database/:resource

Description: Get a list of resources (vns, characters, tags, producers, staff, or traits)

Query Parameters:

  • page: Page number (default: 1)
  • limit: Number of items per page (default: 20, max: 100)
  • sort: Field to sort by
  • order: Sort order (asc or desc)

Get Resource

GET /api/v3/database/:resource/:id

Description: Get details of a specific resource

Create Resource

POST /api/v3/database/:resource

Description: Create a new resource

Body: JSON object with resource details

Update Resource

PUT /api/v3/database/:resource/:id

Description: Update an existing resource

Body: JSON object with updated resource details

Delete Resource

DELETE /api/v3/database/:resource/:id

Description: Delete a specific resource

Backup Database

POST /api/v3/database/backup

Description: Create a database backup

Body: JSON object with optional filename

Restore Database

POST /api/v3/database/restore

Description: Restore the database from a backup

Body: JSON object with filename

Cleanup Database

POST /api/v3/database/cleanup

Description: Perform database cleanup

Query Parameters:

  • type: Cleanup type (vn, tag, producer, staff, character, trait, or all)

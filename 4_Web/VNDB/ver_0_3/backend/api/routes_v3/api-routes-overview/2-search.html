VNDB API v3 Search Routes

Search

GET /api/v3/search

Description: Search for VNs, characters, tags, producers, staff, or traits

Query Parameters:

  • type: Search type (vn, character, tag, producer, staff, trait)
  • query: Search query string
  • fields: Comma-separated list of fields to search in
  • page: Page number (default: 1)
  • limit: Number of items per page (default: 20, max: 100)
  • sort: Field to sort by (default: id)
  • order: Sort order (asc or desc, default: asc)

Response: JSON object with a task_id for asynchronous processing

Get Resource by ID

GET /api/v3/search/:resource/:id

Description: Get details of a specific resource by its ID

Path Parameters:

  • resource: Resource type (vns, characters, tags, producers, staff, traits)
  • id: ID of the resource

Query Parameters:

  • size: Response size (small or large, default: small)

Response: JSON object with a task_id for asynchronous processing

General Endpoints

GET /api/v4

A simple hello world endpoint.
Response: Returns a greeting message.

Tasks API

GET /api/v4/tasks

Retrieve the status of all tasks.
Response: Returns a list of all tasks with their IDs, statuses, and completion dates.

GET /api/v4/tasks/{task_id}

Retrieve the status of a specific task.
Response: Returns the state and status of the specified task. For completed tasks, it includes the result or error message.

GET /api/v4/tasks/{task_id}/result

Retrieve the result of a specific task.
Response: Returns the result of the task if it's completed, or the current state if it's still in progress.

POST /api/v4/tasks

Revoke (cancel) all active tasks.
Response: Returns the number of tasks revoked and the total number of tasks.

POST /api/v4/tasks/{task_id}

Revoke (cancel) a specific task.
Response: Returns a confirmation message that the task has been cancelled.

DELETE /api/v4/tasks

Delete all task results.
Response: Returns the number of task results deleted and the total number of tasks.

DELETE /api/v4/tasks/{task_id}

Delete the result of a specific task.
Response: Returns a confirmation message that the task result has been deleted.

Backups API

GET /api/v4/backups

Retrieve a list of all backups.
Response: Returns a task ID for asynchronous processing.

GET /api/v4/backups/{id}

Retrieve information about a specific backup.
Response: Returns a task ID for asynchronous processing.

POST /api/v4/backups

Create a new backup of the current system state.
Response: Returns a task ID for asynchronous processing.

POST /api/v4/backups/{id}

Restore the system to the state saved in the specified backup.
Response: Returns a task ID for asynchronous processing.

DELETE /api/v4/backups

Response: Returns a task ID for asynchronous processing.

DELETE /api/v4/backups/{id}

Delete a specific backup.
Response: Returns a task ID for asynchronous processing.

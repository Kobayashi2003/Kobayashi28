Resource API Documentation

Common Endpoints

The following endpoints are common to all resource types (VN, Character, Producer, Staff, Tag, Trait).

GET /{resource_type}s

Retrieve a list of resources.

Query Parameters:

ParameterTypeDefaultDescription
pageinteger1Page number for pagination
limitinteger20Number of items per page
sortstring'id'Field to sort by
orderstring'asc'Sort order ('asc' or 'desc')

Returns: A task ID for asynchronous processing.

GET /{resource_type}s/{id}

Retrieve a specific resource by ID.

Returns: A task ID for asynchronous processing.

POST /{resource_type}s

Search for resources.

Request Body:

ParameterTypeDefaultDescription
search_fromstring'local'Where to search from
response_sizestring'small'Size of the response
pageinteger1Page number for pagination
limitinteger20Number of items per page
sortstring'id'Field to sort by
orderstring'asc'Sort order ('asc' or 'desc')

Returns: A task ID for asynchronous processing.

POST /{resource_type}s/{id}

Search for a specific resource by ID.

Returns: A task ID for asynchronous processing.

PUT /{resource_type}s

Update multiple resources.

Returns: A task ID for asynchronous processing.

PUT /{resource_type}s/{id}

Update a specific resource.

Returns: A task ID for asynchronous processing.

PATCH /{resource_type}s

Edit multiple resources.

Request Body:

A list of update objects, each containing the resource ID and fields to update.

Returns: A task ID for asynchronous processing.

PATCH /{resource_type}s/{id}

Edit a specific resource.

Request Body:

An object containing the fields to update.

Returns: A task ID for asynchronous processing.

DELETE /{resource_type}s

Delete all resources of a specific type.

Returns: A task ID for asynchronous processing.

DELETE /{resource_type}s/{id}

Delete a specific resource.

Returns: A task ID for asynchronous processing.

VN-Specific Endpoints

GET /vns/{id}/images

Retrieve images for a specific VN.

Returns: A task ID for asynchronous processing.

GET /vns/{id}/images/{image_id}

Retrieve a specific image for a VN.

Query Parameters:

ParameterTypeDefaultDescription
formatstring'file'Response format ('file' or 'json')

Returns: The image file or a task ID for asynchronous processing.

POST /vns/{id}/images

Upload images for a VN.

Request Body:

Form data with 'files' field containing image files.

Returns: A task ID for asynchronous processing.

PUT /vns/{id}/images

Update all images for a VN.

Returns: A task ID for asynchronous processing.

PUT /vns/{id}/images/{image_id}

Update a specific image for a VN.

Request Body:

Form data with 'file' field containing the image file.

Returns: A task ID for asynchronous processing.

DELETE /vns/{id}/images

Delete all images for a VN.

Returns: A task ID for asynchronous processing.

DELETE /vns/{id}/images/{image_id}

Delete a specific image for a VN.

Returns: A task ID for asynchronous processing.

GET /vns/{id}/savedatas

Retrieve save data for a VN.

Query Parameters:

ParameterTypeDefaultDescription
formatstring'file'Response format ('file' or 'json')

Returns: A zip file containing save data or a task ID for asynchronous processing.

GET /vns/{id}/savedatas/{savedata_id}

Retrieve specific save data for a VN.

Query Parameters:

ParameterTypeDefaultDescription
formatstring'file'Response format ('file' or 'json')

Returns: The save data file or a task ID for asynchronous processing.

POST /vns/{id}/savedatas

Upload save data for a VN.

Request Body:

Form data with 'files' field containing save data files and 'last_modified_{filename}' fields for each file's last modified timestamp.

Returns: A task ID for asynchronous processing.

DELETE /vns/{id}/savedatas

Delete all save data for a VN.

Returns: A task ID for asynchronous processing.

DELETE /vns/{id}/savedatas/{savedata_id}

Delete specific save data for a VN.

Returns: A task ID for asynchronous processing.
